# shiny-octo-goggles
## Repository description  
This repository contains source code and demos for Advanced Game Programming module. Content for each of the relevant weeks are in the respective directories.

### Content overview  

| Week   | Content |
| ------------- | ------------- |
| Week 8  | General 3D computer graphics demos  |
| Week 9  | Samples demonstrating Unity's [Shader Graph](https://docs.unity3d.com/Manual/shader-graph.html)   |
| Week 10  | Unity ShaderLab and HLSL code demonstrating basic shader scripting concepts (Shaders, Sub-Shaders, Passes, Vertex and Fragment shaders)  |
| Week 11  | ShaderLab and HLSL code demonstating various shader models available in Unity's [Universal Render Pipeline (URP)](https://unity.com/features/srp/universal-render-pipeline) |
